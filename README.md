Este repositorio contiene **múltiples proyectos independientes en Java** diseñados como **ejercicios prácticos de refuerzo** de los conceptos fundamentales de **Programación Orientada a Objetos (POO)**.

---

## Objetivos del Reforzamiento

Dominar los pilares de la POO mediante ejemplos claros y progresivos:

| Concepto | Descripción |
|--------|-----------|
| **Clases y Objetos** | Creación, instanciación y encapsulamiento |
| **Constructores** | Sobrecarga, parámetros y valores por defecto |
| **Herencia** | `extends`, reutilización de código, jerarquías |
| **Polimorfismo** | Sobrescritura (`@Override`) y métodos dinámicos |
| **Abstracción** | Clases abstractas e interfaces |
| **Encapsulamiento** | `private`, `public`, `protected`, getters/setters |
| **Sobrecarga** | Mismos nombres, diferentes parámetros |
